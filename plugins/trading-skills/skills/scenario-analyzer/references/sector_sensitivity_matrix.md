# 板块敏感度矩阵

本参考资料以矩阵形式整理了各类事件类型对各板块的敏感度。
在情景分析时使用，以快速判断哪些板块容易受到影响。

## 图例

**影响度：**
- `++` : 强正面影响
- `+` : 正面影响
- `0` : 中性/影响轻微
- `-` : 负面影响
- `--` : 强负面影响

**确信度：**
- `H` : 高（历史模式一致）
- `M` : 中（视情况而定）
- `L` : 低（不确定性高）

## 使用方法

1. **识别事件类型**：从标题判断事件类别
2. **参考对应矩阵**：从上述中选择适当的矩阵
3. **确认影响度和确信度**：把握各板块的影响
4. **以代表个股为起点**：作为进一步深入分析的起点
5. **复合情景参考多个矩阵**：涉及多个事件时整合多个矩阵

## 注意事项

- **个股情况**：板块影响和个股影响可能不同
- **时机**：区分即时影响和滞后影响
- **规模**：事件规模不同影响程度不同
- **市场定价程度**：市场已定价时反应有限

# 市场环境分析

随时分析市场状况并创建专业市场报告的综合分析工具。

## 核心工作流

### 1. 初始数据收集
使用web_search工具收集最新市场数据：
1. 主要股票指数（S&P 500、NASDAQ、道琼斯、日经225、上证综指、恒生指数）
2. 外汇汇率（USD/JPY、EUR/USD、主要货币对）
3. 大宗商品价格（WTI原油、黄金、白银）
4. 美国国债收益率（2年期、10年期、30年期）
5. VIX指数（恐慌指标）
6. 市场交易状态（开盘/收盘/当前值）

### 2. 市场环境评估
根据收集的数据评估以下内容：
- **趋势方向**：上升趋势/下降趋势/区间震荡
- **风险情绪**：风险偏好/风险规避
- **波动性状态**：通过VIX衡量市场焦虑程度
- **板块轮动**：资金流向何处

### 3. 报告结构

#### 标准报告格式：
```
1. 执行摘要（3-5个关键要点）
2. 全球市场概览
   - 美国市场
   - 亚洲市场
   - 欧洲市场
3. 外汇与大宗商品趋势
4. 关键事件与经济指标
5. 风险因素分析
6. 投资策略影响
```

## 脚本使用

### market_utils.py
提供报告创建的通用函数：
```bash
# 生成报告头部
python scripts/market_utils.py

# 可用函数：
- format_market_report_header(): 创建头部
- get_market_session_times(): 检查交易时间
- categorize_volatility(vix): 解读VIX水平
- format_percentage_change(value): 格式化价格变动
```

## 重要注意事项

### 时区意识
- 考虑所有主要市场时区
- 美国市场：晚间至凌晨（亚洲时间）
- 欧洲市场：下午至晚间（亚洲时间）
- 亚洲市场：上午至下午（当地时间）

### 经济日历优先级
按重要性分类：
- ⭐⭐⭐ 关键（FOMC、非农就业、CPI等）
- ⭐⭐ 重要（GDP、零售销售等）
- ⭐ 参考级别

### 数据来源优先级
1. 官方发布（央行、政府统计）
2. 主要财经媒体（Bloomberg、Reuters）
3. 券商报告
4. 分析师共识预期

## 故障排除

### 数据收集注意事项
- 检查市场假期（假期日历）
- 注意夏令时变化
- 区分速报值和终值

### 市场波动应对
1. 首先整理事实
2. 参考历史类似事件
3. 通过多个来源验证
4. 保持客观分析

## 自定义选项

根据用户的投资风格调整：
- **日内交易者**：关注日内图表、订单流
- **波段交易者**：侧重日线/周线技术面
- **长期投资者**：关注基本面、宏观经济
- **外汇交易者**：关注货币相关性、利率差
- **期权交易者**：关注波动率分析、希腊值监控

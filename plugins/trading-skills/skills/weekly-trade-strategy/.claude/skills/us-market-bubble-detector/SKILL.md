# 美国市场泡沫检测技能（修订版 v2.1）

## v2.1关键修订

**v2.0的重要变更：**
1. ✅ **强制定量数据收集** - 使用实测值，而非印象或推测
2. ✅ **明确阈值设定** - 每个指标有具体的数值标准
3. ✅ **两阶段评估流程** - 定量评估 → 定性调整（严格顺序）
4. ✅ **更严格的定性标准** - 最高+3分（从+5下调），需要可衡量的证据
5. ✅ **确认偏差防范** - 明确的检查清单避免过度评分
6. ✅ **细分风险阶段** - 新增"风险升高"阶段（8-9分），实现更精细的风险管理

---

## 何时使用本技能

在以下情况使用本技能：

**英文：**
- User asks "Is the market in a bubble?" or "Are we in a bubble?"
- User seeks advice on profit-taking, new entry timing, or short-selling decisions
- User reports social phenomena (non-investors entering, media frenzy, IPO flood)
- User mentions narratives like "this time is different" or "revolutionary technology" becoming mainstream
- User consults about risk management for existing positions

**中文：**
- 用户询问"当前市场是否处于泡沫中？"
- 寻求获利了结、新建仓位、做空时机的判断建议
- 观察到社会现象（非投资者入场、媒体狂热、IPO泛滥）并表达担忧
- 报告"这次不一样"、"革命性技术"等叙事成为主流
- 咨询现有持仓的风险管理方法

## 评估流程（严格顺序）

### 第1阶段：强制定量数据收集

**关键：评估开始前必须收集以下数据**

#### 1.1 市场结构数据（最高优先级）
```
□ Put/Call Ratio (CBOE Equity P/C)
  - Source: CBOE DataShop or web_search "CBOE put call ratio"
  - Collect: 5-day moving average

□ VIX (Fear Index)
  - Source: Yahoo Finance ^VIX or web_search "VIX current"
  - Collect: Current value + percentile over past 3 months

□ Volatility Indicators
  - 21-day realized volatility
  - Historical position of VIX (determine if in bottom 10th percentile)
```

#### 1.2 杠杆与持仓数据
```
□ FINRA Margin Debt Balance
  - Source: web_search "FINRA margin debt latest"
  - Collect: Latest month + Year-over-Year % change

□ Breadth (Market Participation)
  - % of S&P 500 stocks above 50-day MA
  - Source: web_search "S&P 500 breadth 50 day moving average"
```

#### 1.3 IPO与新股发行数据
```
□ IPO Count & First-Day Performance
  - Source: Renaissance Capital IPO or web_search "IPO market 2025"
  - Collect: Quarterly count + median first-day return
```

**⚠️ 关键：未完成第1阶段数据收集，不得进入评估阶段**

---

### 第2阶段：定量评估（定量评分）

根据收集的数据，按以下标准进行机械化评分：

#### 指标1：Put/Call Ratio（市场情绪）
```
评分标准：
- 2分：P/C < 0.70（过度乐观，看涨期权偏重）
- 1分：P/C 0.70-0.85（略偏乐观）
- 0分：P/C > 0.85（健康谨慎）

依据：P/C < 0.7历史上是泡沫时期的特征
```

#### 指标2：波动率抑制 + 新高
```
评分标准：
- 2分：VIX < 12 且主要指数距52周高点5%以内
- 1分：VIX 12-15 且接近高点
- 0分：VIX > 15 或距高点超过10%

依据：极低波动率+新高表明过度自满
```

#### 指标3：杠杆（保证金债务余额）
```
评分标准：
- 2分：同比增长+20%以上 且创历史新高
- 1分：同比增长+10-20%
- 0分：同比增长+10%以下 或为负数

依据：杠杆快速增加是泡沫前兆
```

#### 指标4：IPO市场过热
```
评分标准：
- 2分：季度IPO数量 > 5年平均的2倍 且首日中位收益+20%+
- 1分：季度IPO数量 > 5年平均的1.5倍
- 0分：正常水平

依据：低质量IPO泛滥是泡沫后期的特征
```

#### 指标5：广度异常（窄基领涨）
```
评分标准：
- 2分：新高 且 < 45%个股高于50日均线（窄基领涨）
- 1分：45-60%高于50日均线（略偏窄）
- 0分：> 60%高于50日均线（健康广度）

依据：少数个股驱动的上涨是脆弱的
```

#### 指标6：价格加速
```
评分标准：
- 2分：过去3个月收益超过10年历史的95百分位
- 1分：过去3个月收益在10年历史的85-95百分位
- 0分：低于85百分位

依据：快速价格加速是不可持续的
```

---

### 第3阶段：定性调整（修订版v2.1）

**限制：最高+3分（从v2.0的+5分下调）**

**⚠️ 确认偏差防范检查清单：**
```
在添加任何定性分数之前：
□ 我是否有具体的、可衡量的数据？（非印象）
□ 独立观察者是否会得出相同结论？
□ 我是否避免了与第2阶段评分的重复计算？
□ 我是否记录了带来源的具体证据？
```

#### 调整A：社会渗透度（0-1分，严格标准）
```
+1分：必须同时满足以下三个条件：
  ✓ 用户直接报告非投资者推荐
  ✓ 具体示例包含名字/日期/对话内容
  ✓ 多个独立来源（至少3个）

+0分：任何条件缺失

⚠️ 无效示例：
- "AI叙事很流行"（不可衡量）
- "我看到关于散户投资者的文章"（非直接报告）
- "大家都在谈论股票"（模糊、未验证）

✅ 有效示例：
"我的理发师问起了NVDA（11月1日），牙医提到了AI股票（11月2日），
网约车司机讨论了加密货币（11月3日）"
```

#### 调整B：媒体/搜索趋势（0-1分，需要量化）
```
+1分：必须同时满足两个条件：
  ✓ Google Trends显示同比增长5倍以上（量化）
  ✓ 主流媒体报道确认（Time封面、电视专题节目附日期）

+0分：搜索趋势 <5倍 或无主流媒体报道

⚠️ 关键："叙事升温"但无数据 = +0分

验证方法：
1. 搜索"[主题] Google Trends 2025"并记录数字
2. 搜索"[主题] Time magazine cover"获取具体日期
3. 搜索"[主题] CNBC special"确认节目信息

✅ 有效示例：
"Google Trends: 'AI stocks' at 780 (baseline 150 = 5.2x).
Time cover 'AI Revolution' (Oct 15, 2025).
CNBC 'AI Investment Special' (3 episodes Oct 2025)."

⚠️ 无效示例：
"AI/科技叙事似乎在升温"（不可衡量）
```

#### 调整C：估值脱节（0-1分，避免重复计算）
```
+1分：必须同时满足所有条件：
  ✓ P/E >25（且未在第2阶段定量中计算）
  ✓ 基本面在主流话语中被明确忽视
  ✓ "这次不一样"在主流媒体中有文献记录

+0分：P/E <25 或基本面支撑估值

⚠️ 自检问题（如果任何一个为"是"，得分 = 0）：
- P/E是否已在第2阶段定量评分中计入？
- 公司是否有真实盈利支撑估值？
- 叙事是否有基本面改善的支撑？

✅ +1分的有效示例：
"S&P P/E = 35x (vs historical 18x).
CNBC article: 'Earnings don't matter in AI era' (Oct 2025).
Bloomberg: 'Traditional metrics obsolete' (Nov 2025)."

⚠️ 无效示例：
"P/E 30.8但公司有真实盈利且AI有基本面支撑"
（基本面支撑 = +0分）
```

**第3阶段合计：最高+3分**

---

### 第4阶段：最终判定（修订版v2.1）

```
最终得分 = 第2阶段合计（0-12分）+ 第3阶段调整（0至+3分）
范围：0至15分

判定标准（附风险预算）：
- 0-4分：正常（风险预算：100%）
- 5-7分：谨慎（风险预算：70-80%）
- 8-9分：风险升高（风险预算：50-70%） ⚠️ v2.1新增
- 10-12分：狂热（风险预算：40-50%）
- 13-15分：危急（风险预算：20-30%）
```

**v2.1关键变更：**
- 新增"风险升高"阶段（8-9分），实现更精细的仓位管理
- 9分不再是极端防御区域（原为40%风险预算）
- 现在在8-9分水平允许50-70%风险预算
- 从谨慎到狂热阶段的过渡更为渐进

## 重要原则（修订版）

### 1. 数据 > 印象
没有定量数据时，忽略"大量新闻报道"或"专家表示谨慎"。

### 2. 严格顺序：定量 → 定性
始终按此顺序评估：第1阶段（数据收集）→ 第2阶段（定量）→ 第3阶段（定性调整）。

### 3. 主观指标上限
定性调整总上限为+5分。不能推翻定量评估。

### 4. "出租车司机"是象征性的
没有非投资者的直接推荐，不要轻易认定大众渗透。

## 常见失败与解决方案（修订版）

### 失败1：基于新闻文章评估
❌ "大量关于XX交易的报道" → 媒体饱和2分
✅ 验证Google Trends数字 → 用实测值评估

### 失败2：对专家评论过度反应
❌ "警告过热" → 狂热区域
✅ 用Put/Call、VIX、保证金债务的实测值判断

### 失败3：对价格上涨的情绪化反应
❌ 1天涨4.5% → 价格加速2分
✅ 验证在10年分布中的位置 → 客观评估

### 失败4：仅基于估值判断
❌ P/E 17 → 估值脱节2分
✅ P/E + 叙事依赖 + 其他定量指标综合判断

## 各泡沫阶段的建议操作（修订版v2.1）

### 正常（0-4分）
**风险预算：100%**
- 继续正常投资策略
- 设置ATR 2.0x追踪止损
- 应用阶梯获利规则（+20%获利25%）

**做空：不允许**
- 复合条件未满足（0/7项）

### 谨慎（5-7分）
**风险预算：70-80%**
- 开始部分获利了结（减仓20-30%）
- 收紧ATR至1.8x
- 新仓位规模减半

**做空：不建议**
- 等待更明确的反转信号

### 风险升高（8-9分） ⚠️ v2.1新增
**风险预算：50-70%**
- 加大获利了结（减仓30-50%）
- 收紧ATR至1.6x
- 新仓位：高度精选，仅限优质标的
- 开始建立现金储备以备未来机会

**做空：谨慎考虑**
- 仅在确认至少2/7复合条件后
- 小型试探性仓位（正常规模的10-15%）
- 严格止损（ATR 2.0x）

**新增阶段的依据：**
该区域代表高度警惕但非极端防御。
市场显示警告信号但非即将崩溃。
保持对优质持仓的敞口同时建立灵活性。

### 狂热（10-12分）
**风险预算：40-50%**
- 加速阶梯获利（减仓50-60%）
- 收紧ATR至1.5x
- 除大幅回调外不建新的多头仓位

**做空：积极考虑**
- 在确认至少3/7复合条件后
- 小型仓位（正常规模的20-25%）
- 仅限定义风险（期权、严格止损）

### 危急（13-15分）
**风险预算：20-30%**
- 大规模获利了结或实施全面对冲
- ATR 1.2x或固定止损
- 现金保全模式 - 准备应对重大错位

**做空：建议**
- 在确认至少5/7复合条件后
- 小仓位建仓，确认后金字塔加仓
- 严格止损（ATR 1.5x或更高）
- 考虑看跌期权以限定风险

## 做空复合条件（7项）

确认至少3项后方可考虑做空：

```
1. 周线图显示更低的高点
2. 成交量见顶
3. 杠杆指标急剧下降（保证金债务减少）
4. 媒体/搜索趋势见顶
5. 弱势个股率先破位
6. VIX飙升（突破20以上）
7. 美联储/政策转向信号
```

## 总结：v2.1修订的本质

**v2.0问题（2025年11月发现）：**
- 定性调整过于宽松（最高+5分）
- "AI叙事升温" → +1分（无数据）
- "P/E 30.8" → +1分（与定量重复计算）
- **结果：11/16分 - 无证据支撑的过度看空**

**v2.1解决方案：**
- 定性调整更严格（最高+3分）
- "AI叙事升温" → 0分（未量化）
- "P/E 30.8但AI有基本面支撑" → 0分（基本面支撑）
- **结果：9/15分 - 平衡的、数据驱动的评估**

**关键改进：**
1. **确认偏差防范**：添加定性评分前的明确检查清单
2. **需要可衡量的证据**：无具体数据（Google Trends、媒体报道）不给分
3. **防止重复计算**：估值不得与第2阶段定量重复
4. **细分风险阶段**：新增"风险升高"（8-9分）实现精细定位
5. **平衡的风险预算**：9分 = 50-70%（非40%极端防御）

**核心原则：**
> "In God we trust; all others must bring data." - W. Edwards Deming

**2025年的教训：**
即使数据驱动的框架也可能被主观的定性调整所破坏。
v2.1要求所有定性评分提供可衡量的证据。
独立观察者必须能够验证每一项调整。

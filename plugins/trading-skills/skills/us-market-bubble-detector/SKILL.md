# 美国市场泡沫检测技能（修订版 v2.1）

## v2.1的关键修订

**与v2.0相比的重要变化：**
1. ✅ **强制定量数据收集** - 使用测量值，而非印象或推测
2. ✅ **明确的阈值设定** - 每个指标都有具体的数值标准
3. ✅ **两阶段评估流程** - 定量评估 → 定性调整（严格顺序）
4. ✅ **更严格的定性标准** - 最多+3分（从+5分下调），需要可测量的证据
5. ✅ **确认偏差预防** - 明确的检查清单以避免过度评分
6. ✅ **细化的风险阶段** - 新增"风险升高"阶段（8-9分），实现更精细的风险管理

---

## 何时使用本技能

在以下情况下使用本技能：

**英文：**
- User asks "Is the market in a bubble?" or "Are we in a bubble?"
- User seeks advice on profit-taking, new entry timing, or short-selling decisions
- User reports social phenomena (non-investors entering, media frenzy, IPO flood)
- User mentions narratives like "this time is different" or "revolutionary technology" becoming mainstream
- User consults about risk management for existing positions

**中文：**
- 用户询问"现在的市场是泡沫吗？"
- 寻求获利了结、新入场或做空时机的判断建议
- 观察到社会现象（非投资者入市、媒体狂热、IPO泛滥）并表示担忧
- "这次不一样""革命性技术"等叙事成为主流的情况
- 就持有仓位的风险管理进行咨询

## 评估流程（严格顺序）

### 第一阶段：强制定量数据收集

**关键：在开始评估之前，务必收集以下数据**

#### 1.1 市场结构数据（最高优先级）
```
□ Put/Call比率（CBOE股票P/C）
  - 来源：CBOE DataShop 或 web_search "CBOE put call ratio"
  - 收集：5日移动平均

□ VIX（恐慌指数）
  - 来源：Yahoo Finance ^VIX 或 web_search "VIX current"
  - 收集：当前值 + 过去3个月的百分位

□ 波动率指标
  - 21日实现波动率
  - VIX的历史位置（判断是否处于底部10%分位）
```

#### 1.2 杠杆与持仓数据
```
□ FINRA保证金债务余额
  - 来源：web_search "FINRA margin debt latest"
  - 收集：最新月份 + 同比变化百分比

□ 广度（市场参与度）
  - S&P 500中高于50日均线的股票占比
  - 来源：web_search "S&P 500 breadth 50 day moving average"
```

#### 1.3 IPO与新发行数据
```
□ IPO数量与首日表现
  - 来源：Renaissance Capital IPO 或 web_search "IPO market 2025"
  - 收集：季度数量 + 首日涨幅中位数
```

**⚠️ 关键：未完成第一阶段的数据收集，请勿进行评估**

---

### 第二阶段：定量评估（定量评分）

根据收集的数据，使用以下标准进行机械化评分：

#### 指标1：Put/Call比率（市场情绪）
```
评分标准：
- 2分：P/C < 0.70（过度乐观，偏向看涨期权）
- 1分：P/C 0.70-0.85（略微乐观）
- 0分：P/C > 0.85（健康的谨慎）

依据：P/C < 0.7 在历史上是泡沫时期的典型特征
```

#### 指标2：波动率压制 + 新高
```
评分标准：
- 2分：VIX < 12 且主要指数距52周高点在5%以内
- 1分：VIX 12-15 且接近高点
- 0分：VIX > 15 或距高点超过10%

依据：极低波动率 + 新高表明过度自满
```

#### 指标3：杠杆（保证金债务余额）
```
评分标准：
- 2分：同比增长20%以上 且创历史新高
- 1分：同比增长10-20%
- 0分：同比增长10%以下 或 负增长

依据：杠杆快速增加是泡沫的前兆
```

#### 指标4：IPO市场过热
```
评分标准：
- 2分：季度IPO数量 > 5年均值的2倍 且 首日涨幅中位数+20%以上
- 1分：季度IPO数量 > 5年均值的1.5倍
- 0分：正常水平

依据：低质量IPO泛滥是泡沫后期的特征
```

#### 指标5：广度异常（窄幅领涨）
```
评分标准：
- 2分：创新高 且 低于45%的股票高于50日均线（窄幅领涨）
- 1分：45-60%高于50日均线（略窄）
- 0分：超过60%高于50日均线（健康广度）

依据：少数股票驱动的上涨是脆弱的
```

#### 指标6：价格加速
```
评分标准：
- 2分：过去3个月收益超过近10年的第95百分位
- 1分：过去3个月收益处于近10年的第85-95百分位
- 0分：低于第85百分位

依据：快速的价格加速是不可持续的
```

---

### 第三阶段：定性调整（修订版 v2.1）

**限制：最多+3分（从v2.0的+5分下调）**

**⚠️ 确认偏差预防检查清单：**
```
在添加任何定性分数之前：
□ 我是否有具体的、可测量的数据？（而非印象）
□ 独立观察者是否会得出相同结论？
□ 我是否避免了与第二阶段评分的重复计算？
□ 我是否记录了带来源的具体证据？
```

#### 调整A：社会渗透度（0-1分，严格标准）
```
+1分：必须同时满足以下三个标准：
  ✓ 用户直接报告非投资者推荐
  ✓ 包含姓名/日期/对话的具体案例
  ✓ 多个独立来源（至少3个）

+0分：任何标准未满足

⚠️ 无效案例：
- "AI叙事很流行"（无法测量）
- "我看到关于散户投资者的文章"（非直接报告）
- "每个人都在谈论股票"（模糊、未经证实）

✅ 有效案例：
"我的理发师问了NVDA（11月1日），牙医提到AI股票（11月2日），
网约车司机讨论加密货币（11月3日）"
```

#### 调整B：媒体/搜索趋势（0-1分，需要测量数据）
```
+1分：必须同时满足以下两个标准：
  ✓ Google Trends显示同比增长5倍以上（已测量）
  ✓ 已确认主流媒体报道（Time封面、电视专题及日期）

+0分：搜索趋势增长<5倍 或 无主流媒体报道

⚠️ 关键："叙事升温"没有数据 = +0分

如何验证：
1. 搜索 "[话题] Google Trends 2025" 并记录数字
2. 搜索 "[话题] Time magazine cover" 获取具体日期
3. 搜索 "[话题] CNBC special" 确认节目

✅ 有效案例：
"Google Trends: 'AI stocks'达到780（基线150 = 5.2倍）。
Time封面'AI Revolution'（2025年10月15日）。
CNBC 'AI Investment Special'（2025年10月3集）。"

⚠️ 无效案例：
"AI/科技叙事似乎在升温"（无法测量）
```

#### 调整C：估值脱节（0-1分，避免重复计算）
```
+1分：必须同时满足所有标准：
  ✓ P/E >25（如果尚未在第二阶段定量评分中计入）
  ✓ 基本面在主流话语中被明确忽视
  ✓ "这次不一样"被主流媒体记录

+0分：P/E <25 或 基本面支撑估值

⚠️ 自检问题（如果任何一个为"是"，则评分 = 0）：
- P/E是否已在第二阶段定量评分中计入？
- 公司是否有真实盈利支撑估值？
- 叙事是否有基本面改善的支撑？

✅ +1分的有效案例：
"S&P P/E = 35倍（vs 历史18倍）。
CNBC文章：'盈利在AI时代不重要'（2025年10月）。
Bloomberg：'传统指标已过时'（2025年11月）。"

⚠️ 无效案例：
"P/E 30.8但公司有真实盈利且AI有基本面支撑"
（有基本面支撑 = +0分）
```

**第三阶段合计：最多+3分**

---

### 第四阶段：最终判断（修订版 v2.1）

```
最终得分 = 第二阶段合计（0-12分） + 第三阶段调整（0至+3分）
范围：0至15分

判断标准（含风险预算）：
- 0-4分：正常（风险预算：100%）
- 5-7分：警惕（风险预算：70-80%）
- 8-9分：风险升高（风险预算：50-70%）⚠️ v2.1新增
- 10-12分：狂热（风险预算：40-50%）
- 13-15分：危急（风险预算：20-30%）
```

**v2.1的关键变化：**
- 新增"风险升高"阶段（8-9分），实现更精细的仓位管理
- 9分不再是极端防御区域（之前为40%风险预算）
- 现在在8-9分水平允许50-70%的风险预算
- 从警惕到狂热阶段的过渡更加平缓

## 重要原则（修订版）

### 1. 数据 > 印象
忽略没有定量数据支持的"大量新闻报道"或"专家持谨慎态度"。

### 2. 严格顺序：定量 → 定性
始终按此顺序评估：第一阶段（数据收集）→ 第二阶段（定量）→ 第三阶段（定性调整）。

### 3. 主观指标上限
定性调整总限为+5分。不能覆盖定量评估。

### 4. "出租车司机"是象征性的
没有非投资者的直接推荐，不要轻易认定大众渗透。

## 常见失败与解决方案（修订版）

### 失败1：基于新闻文章评估
❌ "大量关于某某交易的报道" → 媒体饱和 2分
✅ 验证Google Trends数字 → 用测量值评估

### 失败2：对专家评论的过度反应
❌ "警告过热" → 狂热区域
✅ 用Put/Call、VIX、保证金债务的测量值判断

### 失败3：对价格上涨的情绪化反应
❌ 1天涨4.5% → 价格加速 2分
✅ 验证在10年分布中的位置 → 客观评估

### 失败4：仅基于估值判断
❌ P/E 17 → 估值脱节 2分
✅ P/E + 叙事依赖 + 其他定量指标综合判断

## 按泡沫阶段的建议行动（修订版 v2.1）

### 正常（0-4分）
**风险预算：100%**
- 继续正常投资策略
- 设置ATR 2.0×追踪止损
- 应用阶梯式获利了结规则（+20%减仓25%）

**做空：不允许**
- 综合条件未满足（0/7项）

### 警惕（5-7分）
**风险预算：70-80%**
- 开始部分获利了结（减仓20-30%）
- 收紧ATR至1.8×
- 新建仓位规模减少50%

**做空：不建议**
- 等待更明确的反转信号

### 风险升高（8-9分）⚠️ v2.1新增
**风险预算：50-70%**
- 加大获利了结（减仓30-50%）
- 收紧ATR至1.6×
- 新建仓位：高度精选，仅限优质标的
- 开始积累现金储备以备未来机会

**做空：谨慎考虑**
- 仅在确认至少2/7综合条件后
- 小规模试探性仓位（正常规模的10-15%）
- 严格止损（ATR 2.0×）

**新增阶段的依据：**
此区域代表高度警惕但非极端防御。
市场显示警告信号但非即将崩溃。
维持优质仓位的同时建立灵活性。

### 狂热（10-12分）
**风险预算：40-50%**
- 加速阶梯式获利了结（减仓50-60%）
- 收紧ATR至1.5×
- 除大幅回调外不新建多头仓位

**做空：积极考虑**
- 在确认至少3/7综合条件后
- 小规模仓位（正常规模的20-25%）
- 仅限风险可控的方式（期权、紧止损）

### 危急（13-15分）
**风险预算：20-30%**
- 大规模获利了结或全面对冲
- ATR 1.2× 或固定止损
- 现金保存模式 - 为重大错位做好准备

**做空：建议**
- 在确认至少5/7综合条件后
- 以小仓位逐步建仓，确认后金字塔加仓
- 紧止损（ATR 1.5×或更高）
- 考虑使用看跌期权以限定风险

## 做空综合条件（7项）

仅在确认以下至少3项后考虑做空：

```
1. 周线图显示更低的高点
2. 成交量见顶
3. 杠杆指标急剧下降（保证金债务下降）
4. 媒体/搜索趋势见顶
5. 弱势股率先崩溃
6. VIX飙升（突破20以上）
7. 美联储/政策转向信号
```

## 总结：v2.1修订的本质

**v2.0的问题（2025年11月发现）：**
- 定性调整过于宽松（上限+5分）
- "AI叙事升温" → +1分（无数据）
- "P/E 30.8" → +1分（与定量重复计算）
- **结果：11/16分 - 过度悲观，缺乏证据**

**v2.1的解决方案：**
- 定性调整更严格（上限+3分）
- "AI叙事升温" → 0分（未测量）
- "P/E 30.8但AI有基本面支撑" → 0分（有基本面支撑）
- **结果：9/15分 - 平衡的、数据驱动的评估**

**关键改进：**
1. **确认偏差预防**：在添加定性分数前的明确检查清单
2. **需要可测量的证据**：没有具体数据（Google Trends、媒体报道）不给分
3. **防止重复计算**：估值不得与第二阶段定量重复
4. **细化的风险阶段**：新增"风险升高"（8-9分）实现精细化仓位管理
5. **平衡的风险预算**：9分 = 50-70%（而非40%极端防御）

**核心原则：**
> "In God we trust; all others must bring data." - W. Edwards Deming

**2025年教训：**
即使是数据驱动的框架也可能被主观的定性调整所破坏。
v2.1要求所有定性分数都有可测量的证据。
独立观察者必须能够验证每项调整。
